# InJoy Beauty Landing Page

This repository contains the static landing page for Inclusive Beauty's services.

## Local development

Serve the site with any static file server. For example:

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Testing

Since this is a static page, manual verification is performed by loading the site in a browser and confirming that the navigation, hero section, and footer render as expected.
